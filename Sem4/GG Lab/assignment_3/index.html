Typography Anatomy

Baseline

The baseline is the transparent line where all letters sit. It serves as a reference point for aligning text.

X-height

The x-height is the distance between the baseline of a line of type and the tops of the main body of lowercase letters. The baseline does not include ascenders and descenders. It is determined by the height of the letter "x".

Ascender

An ascender is the upward stem part of lowercase letters that extends beyond the meanline and x-height of a font.

Examples: b, d, k, h, l, t

Descender

A descender is the 'tail' or downward portion of a letter that extends below the baseline. The descender's height is calculated from the baseline to the end of the font descender line.

Examples: j, g, y, p, q

Stem

The stem is the main vertical stroke in upright characters. When a letter has no verticals, like a capital A or V, the first diagonal stroke is considered the stem.

Counter

The counter in typography refers to the space within a letter, either partially or fully enclosed.

  • Closed Counter: A fully enclosed space within a letter (e.g., a, b, d, g, o, p, q).
  • Open Counter: A partially enclosed space that remains open (e.g., c, f, h, m, n, u).

Terminal

A terminal refers to the end of a stroke or a serif in a letterform. When a letter doesn't have a serif, the end of the stroke is called a terminal.

Ligature

Ligatures are used to join two characters together for better readability. For example, in the combination of "f" and "i" or "f" and "l," ligatures help prevent overlapping or misreading.

Kerning

Kerning refers to the adjustment of spacing between two individual characters in a word or text block. Proper kerning enhances readability and ensures harmonious letter spacing.

Leading

Leading is the space between baselines of consecutive lines of text. Leading and x-height directly affect how text appears in a paragraph.

Typeface Classification

Type faces are based on design characteristics and the intentions why they are used.

Serif

They are the small strokes that are used at the end of the letter.

Serif font helps to improve the readability of the text.

Serif fonts are often associated with tradition, reliability, and professionalism.

Commonly used in:

  • Books, newspaper
  • Formal documentations

Examples:

  • This content is written using Garamond
  • This content is written using Times New Roman
  • This content is written using Roboto Serif

Sans-serif

Sans is derived from German word which means without.

These are the serif font but don't have strikes at the end of word.

It help to give clean, modern and minimalist look to the letters.

Commonly seen in:

  • Websites
  • Digital Branding

Examples:

  • This content is written using Helvetica
  • This content is written using Roboto
  • This content is written using Nunito

Script

These typefaces mimic handwriting, feature flowing.

They are often used for artistic and decorative purpose.

They are generally formal or casual depending on design.

Used in:

  • Wedding invitation
  • Luxury branding

Examples:

  • This content is written using Lobster
  • This content is written using Homemade Apple
  • This content is written using Pacifico

Display

Designed for use in larger sizes.

They are generally used for titles, heading etc.

These kind of text are not suitable for body text.

Used in:

  • Website builder
  • Billboards

Examples:

  • This content is written using Impact
  • This content is written using Cooper Black

Monospace

These are the fonts which have exact same height and width of each font.

Commonly used in:

  • Coding because they help in easy understanding of code and makes it easy to debug the code.
  • Terminals

Typography in Poster Design

Harmonious Typography

Different typefaces working together in harmony to create visual hierarchy and balance.

  • Clear visual hierarchy
  • Complementary fonts
  • Balanced composition

Poor Typography

Single typeface used throughout, creating a monotonous visual experience.

  • Lack of contrast
  • No visual hierarchy
  • Poor readability

Mixed Typography

Multiple typefaces creating visual interest and dynamic composition.

  • Dynamic contrast
  • Varied elements
  • Strong impact
